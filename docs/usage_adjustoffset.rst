adjustoffset
------------

Description:
^^^^^^^^^^^^
	Adjust the offset of a rapidtide delay map.

Inputs:
^^^^^^^
	A rapidtide delay map.

Outputs:
^^^^^^^^
	A new rapidtide delay map, with a different time offset.
